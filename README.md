# Exam Preparation Scheduler App

A simple React Native application to schedule exam preparation.

## Features
- Add exam subjects
- Set exam date
- Daily study hours
- Light theme UI
- No database
- Simple Node.js backend

## Tech Stack
- React Native (Expo)
- Node.js
- Express

## How to Run
### Backend
cd backend  
node server.js  

### Frontend
cd frontend  
npx expo start
